# Gaming-vs-Academic-Performance-1-
🎮📚 ¿Los videojuegos afectan el rendimiento académico? 📊 En este proyecto analicé la relación entre videojuegos, estrés y calificaciones. El hallazgo más interesante fue el impacto del estrés: no se trata solo de cuánto estrés tenemos, sino de cómo lo gestionamos para encontrar un equilibrio entre la vida académica y personal. ⚖️🧠
